## Install `poppler`
* OSX
```
brew install poppler
```
* Ubuntu
```
sudo apt-get install poppler-utils
```

## Usaage

```
python3 ./pdf-extractor.py pdfname.pdf pagenumber
```

* Make selection with mouse drag
* You already have `out.pdf` in the same directory :-)
